### our-medical 简介

    our-medical是由[西安大医集团股份有限公司](http://www.ourunited.com/)前端项目组基于[cornerstone3D](https://www.cornerstonejs.org/)开发，用于显示DICOM医学影像的vue组件（暂不开放源码敬请谅解），部分功能基于公司项目需求实现可能不适合其他开发者使用。

    支持 Typescript。

### 安装

```sh
npm install our-medical 
```

    使用教程可结合examples中的代码查阅docs
